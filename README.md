# stack-test

## Written With
- Ruby

## Getting Started
Before we start, make sure you fit the prerequisites

### Setup
- git clone `https://github.com/ahmedadel56/stack-test.git`
  
### Install
- `bundle install`

### Usage
- `ruby classes/stack.rb`
- `ruby classes/extras.rb`

### Run linters tests
- `rubocop`

### Run Spec tests
- `rspec ./spec`
